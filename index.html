Veterinaria: Sistema de Información, Página Web y Aplicación Móvil

El proyecto "Veterinaria" consistió en el Desarrollo de un sistema integral para una clínica veterinaria que incluye un sistema de información para gestionar expedientes de pacientes, una página web informativa y una aplicación móvil para brindar a los clientes acceso a servicios y recursos veterinarios.

Resultado: El sistema de información permite a la clínica gestionar de manera eficiente los registros de pacientes, citas, historias clínicas y facturación. La página web ofrece información detallada sobre los servicios de la clínica, horarios de atención y contacto, lo que ha aumentado la visibilidad en línea y atraído a nuevos clientes. La aplicación móvil permite a los dueños de mascotas programar citas, acceder a consejos de cuidado y recibir notificaciones importantes, mejorando así la experiencia del cliente y la fidelidad a la clínica veterinaria.

Tecnologías Utilizadas: Backend: Django, Azure SQL Database. Frontend (Página Web): Angular, HTML, CSS. Aplicación Móvil: Ionic. Bases de Datos: PostgreSQL. Servicios en la Nube: Microsoft Azure.

Resultado : El sistema de información permite a la clínica gestionar de manera eficiente los registros de pacientes, citas, historias clínicas y facturación. La página web ofrece información detallada sobre los servicios de la clínica, horarios de atención y contacto, lo que ha aumentado la visibilidad en línea y atraído a nuevos clientes. La aplicación móvil permite a los dueños de mascotas programar citas, acceder a consejos de cuidado y recibir notificaciones importantes, mejorando así la experiencia del cliente y la fidelidad a la clínica veterinaria.

Impacto para el Cliente: Mayor eficiencia en la gestión de pacientes y registros. Aumento en la clientela debido a la visibilidad en línea. Mayor satisfacción de los clientes gracias a la conveniencia de la aplicación móvil. Optimización de los procesos internos de la clínica veterinaria, lo que ha llevado a una mejor atención y cuidado de las mascotas.

Crecimiento Personal Coach Persona - Aplicación Móvil Android

El proyecto "Crecimiento Personal Coach Persona" fue una aplicación móvil para dispositivos Android diseñada para brindar a los usuarios herramientas y recursos para su desarrollo personal y bienestar. La aplicación fue desarrollada con el objetivo de ofrecer un acceso conveniente a contenido educativo y actividades interactivas relacionadas con el crecimiento personal y el coaching personalizado.

Tecnologías Utilizadas: Desarrollo de Aplicación Móvil: Android Studio, Java. Almacenamiento en la Nube: Microsoft Azure para el almacenamiento seguro de datos y contenido. Plataforma Web Complementaria: WordPress con el tema Divi para gestionar el contenido y recursos de la aplicación.

Resultado: La aplicación permitió a los usuarios acceder a una amplia variedad de contenido, incluyendo artículos, videos, ejercicios de reflexión y seguimiento de metas personales. La integración con Azure garantizó la seguridad y disponibilidad de los datos de los usuarios. El uso de WordPress con el tema Divi permitió a los administradores de la aplicación gestionar y actualizar fácilmente el contenido sin necesidad de habilidades técnicas avanzadas.

Impacto para el Cliente: La aplicación recibió una respuesta positiva de los usuarios, quienes informaron un mayor compromiso en su crecimiento personal y bienestar. El cliente experimentó un aumento en la retención de usuarios y una mayor participación en la plataforma, lo que llevó a un impacto positivo en su negocio de coaching personal.
